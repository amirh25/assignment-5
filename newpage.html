Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll()

Question-2: What is the different innerHTML, innerText, textContent?

innerHTML gets or sets content with HTML tags, innerText gets or sets visible text only, and textContent gets or sets all text, including hidden ones.

Question-3: what is the event delegation in the dom?

Event delegation is a technique in the DOM where instead of adding event listeners to each individual element, a single event listener is added to a parent element, and events are handled for its child elements via event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event triggered on an element propagates upward through its ancestor elements, from the target element to the root, allowing parent elements to handle the event
